Project Keystone

1. Purpose: Create flexible framework for user account creation and login using DynamoDB

2. Goals: 
    a. Allow users to register accounts with a username, password, first name, last name, and email address
    b. Allow users to sign into their account via DB authentication 
    c. Allow users to delete their accounts once done
    d. Be able to log in and out via GUI
    e. Allow users to edit their accounts
